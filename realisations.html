L'Histoire de Pourcentage RP

En 2019, une crise économique frappe les États-Unis suite à l'effondrement des chaînes d'import-export. San Andreas, île dépendante du commerce, est dévastée par l'inflation et le chômage.

Merryweather Security profite du chaos pour prendre le contrôle de Los Santos, instaurant une dictature militaire. La population fuit vers le Nord, notamment à Roxwood, territoire indépendant qui résiste avec Paleto Bay contre les forces de Merryweather.

En 2025, malgré les tentatives de reconquête, l'avenir reste incertain pour Roxwood et ses habitants face à la menace persistante de Merryweather.

Bot Discord L'Entrepôt

Bot Discord • 2021-Projet actif • ~15h de développement

Bot permettant de gérer mon serveur Discord l'Entrepôt. Développé avec Discord.py, il offre une gestion complète des rôles, un système de tickets et de multiples fonctionnalités d'administration.

Technologies utilisées
Python Discord.py JSON
Fonctionnalités
  • Système d'auto-rôles avec boutons interactifs pour les langues et catégories
  • Gestion des tickets de support avec création automatique de salons privés
  • Système de vérification des membres avec attribution de rôles
  • Personnalisation des pseudonymes avec sélection de couleurs par boutons
Extrait de code
@bot.command()
async def roles(ctx):
    embed = discord.Embed(
        title="🎭 Rôles Disponibles",
        description=(
            "Sélectionnez un rôle en cliquant sur les boutons ci-dessous :\n\n"
            "🟩 **Minecraft** : Obtenez le rôle Minecraft.\n"
            "🟦 **FiveM** : Obtenez le rôle FiveM.\n"
            "🟪 **Développeur** : Obtenez le rôle Développeur.\n"
            "🪙 **Cryptomonnaie** : Obtenez le rôle Cryptomonnaie.\n\n"
            "Vous pouvez cliquer à nouveau pour retirer le rôle."
        ),
        color=0x00FF00
    )
    
    view = View(timeout=None)
    
    minecraft_button = Button(label="Minecraft", style=discord.ButtonStyle.green, emoji="🟩")
    async def minecraft_callback(interaction):
        role = interaction.guild.get_role(MINECRAFT_ROLE_ID)
        if role in interaction.user.roles:
            await interaction.user.remove_roles(role)
            await interaction.response.send_message("Le rôle Minecraft a été retiré.", ephemeral=True)
        else:
            await interaction.user.add_roles(role)
            await interaction.response.send_message("Le rôle Minecraft a été attribué.", ephemeral=True)
    
    minecraft_button.callback = minecraft_callback
    view.add_item(minecraft_button)

@bot.event
async def on_message(message):
    if isinstance(message.channel, discord.DMChannel) and message.author != bot.user:
        guild = bot.get_guild(SERVER_ID)
        category = discord.utils.get(guild.categories, id=TICKET_CATEGORY_ID)
        existing_channel = await find_existing_ticket(message.author, guild, category)

        if existing_channel:
            await existing_channel.send(f"Nouveau message de {message.author.mention} :\n> {message.content}")
            
            for attachment in message.attachments:
                if any(attachment.filename.lower().endswith(ext) for ext in ['.png', '.jpg', '.jpeg', '.webp']):
                    await existing_channel.send(file=await attachment.to_file())
            
            if str(message.author.id) not in open_tickets:
                open_tickets[str(message.author.id)] = existing_channel.id
                save_tickets()
        else:
            ticket_channel = await guild.create_text_channel(f'ticket-{message.author.name}', category=category)
            open_tickets[str(message.author.id)] = ticket_channel.id
            save_tickets()
